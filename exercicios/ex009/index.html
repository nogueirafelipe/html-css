Trabalhando com Listas

Listas Ordenadas

Listas ordenadas são aquelas em que a ordem dos itens é importante. Um exemplo de lista ordenada:

  1. Escovar os dentes
  2. Tomar café
  3. Lavar a louça
  4. Responder e-mails

É possível personalizar a formatação da lista ordenada. Na lista acima os itens estão listados em ordem numeral, é possível fazer com que eles sejam listados em algarismos romanos (maiúsculos ou minúsculos) ou em letras (também maiúsculas ou minúsculas) usando o parâmetro type. Por exemplo:

  1. Escovar os dentes
  2. Tomar café
  3. Lavar a louça
  4. Responder e-mails
  1. Messi
  2. C. Ronaldo
  3. Lewandoski
  4. Neymar
  5. Haaland

Além disso, também é possível indicar o início da contagem usando o parâmetro start. Por exemplo:

  1. Lavar a louça
  2. Responder e-mails
  3. Estudar HTML5
  4. Pausa
  5. Voltar a estudar

Listas Não Ordenadas

As listas não ordenadas são aquelas em que a sequência não importa. Um exemplo de lista não ordenada:

  • Arroz
  • Feijão
  • Pão
  • Amaciante
  • Detergente

Também é possível personalizar a lista não ordenada usando o parâmetro type, para o qual os valores são: disc, circle e square. Exemplo:

  • Arroz
  • Feijão
  • Pão
  • Amaciante
  • Detergente
  • Arroz
  • Feijão
  • Pão
  • Amaciante
  • Detergente

Misturando as coisas

É possível configurar listas dentro de outras listas. Além disso, também podemos misturar os dois tipos de listas, ordenadas com não ordenadas. Exemplos:

Lista dos meus jogos favoritos:

  1. Antigos
    1. Super Mario World
    2. The Legend of Zelda: A Link To The Past
    3. Pagemaster
    4. Super Bomberman 3
  2. Mais recentes
    1. God of War 2
    2. Batman Arkham City
    3. Assassins Creed Black Flag
    4. GTA San Andreas

Agora misturando os dois tipos de lista

Lista de Definições

É como se fosse um dicionário, temos os termos e suas descrições. É uma lista sem demarcadores, mas bem útil em alguns casos.

HTML
Linguagem de marcação utilizada para criar o conteúdo de um site.
CSS
Linguagem de marcação para a especificação de estilos em um site.
JavaScript
Linguagem de programação para cirar a interatividade em um site.
