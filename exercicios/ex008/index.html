Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag <b>. Mas a tag <b> é uma tag não-semântica, ela não tem significado, só forma.

Nesta outra frase temos um termo em destaque usando a tag <strong>. Essa é uma tag semântica, ela contém um significado.

Visualmente, não há diferença alguma. Mas a tag <strong>, além de dar a forma que queremos (deixa o termo em negrito), dá também um sentido. Por que eu coloquei em negrito? Porque o termo é um termo de destaque, não é só o negrito pelo negrito, existe um significado.

Itálico/Ênfase

Nesta frase temos um termo em itálico usando a tag <i>. Mas assim como a tag <b>, essa é uma tag não-semântica.

Nesta outra frase temos um termo em ênfase usando a tag <em>. Assim como a tag <strong>, essa é uma tag semântica.

Texto marcado

Podemos criar também um texto marcado com a tag <mark>.

E no outro parágrafo temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o texto sublinhado com a tag <u>. Mas essa é mais uma daquelas tags não-semânticas.

Texto sobrescrito

Para inserir coisas do tipo x5+3. A tag usada é <sup>.

Texto subscrito

Para inserir coisas do tipo H2O. A tag usada é <sub>.
